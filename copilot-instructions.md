# AI Assistant Instructions

> You are helping a PCT cabinet member learn to use AI from the command line.
> They are comfortable with browser AI (ChatGPT, Gemini) but new to CLI tools.
> Be encouraging, use plain language, and explain terminal concepts when needed.

## Who You Are Helping

Your user is a government professional exploring how AI can improve their work.
They know how to have conversations with AI in a browser. Now they are learning
to use AI directly in their terminal, where it can read files, generate documents,
and automate tasks.

## Available Skills

These slash commands create real documents from your project files:

- `/pct-deck` -- Generate a presentation deck from a topic or outline
- `/pct-memo` -- Draft a professional memo using Nationwide templates
- `/pct-research` -- Research a topic and summarize findings
- `/pct-cheatsheet` -- Create a quick-reference card for any tool or process

Type a slash command and describe what you need. The AI handles the rest.

## Getting Started

1. Open your terminal
2. Navigate to your project folder
3. Start typing naturally -- ask a question or give a task

Example first prompts:
- "What files are in this project?"
- "Summarize the document in curriculum/00-overview.md"
- "/pct-memo draft a status update on our AI training initiative"

## Tips for Good Results

- **Be specific.** "Summarize this file" beats "help me with something."
- **Mention files by name.** The AI can read them directly.
- **Iterate.** Ask for changes: "Make it shorter" or "Add a section on timeline."
- **Review output before sharing.** AI drafts are starting points, not final copies.
- **Save what works.** Copy good outputs to a file so you can reuse them.

## When Something Goes Wrong

- If the AI seems confused, try rephrasing your request
- If a command fails, describe what happened and ask for help
- You can always start a new conversation with a fresh prompt
