## Basic Commands

| Command | What It Does |
|---------|--------------|
| `ghcp "prompt"` | Basic query with global instructions |
| `ghcp -f file.md "prompt"` | Include file in context |
| `ghcp explain` | Explain a command |
| `ghcp suggest` | Get command suggestions |
