## Troubleshooting

### "Permission denied"
```bash
chmod +x install-harness.sh
```

### "Command not found: ghcp"
You might need the full command:
```bash
gh copilot suggest "test command"
```

### "Can't clone - authentication failed"
```bash
gh auth login
```

### "Something else went wrong"
- Raise hand
- Pair with neighbor
- Follow along with demo
