## Step 2: Run Setup

```bash
# Make setup script executable
chmod +x setup.sh

# Run setup
./setup.sh
```

**What this does:**
- Creates symlinks for global instructions
- Sets up skill directory
- Configures any needed environment

**Checkpoint:** Script completes without errors
