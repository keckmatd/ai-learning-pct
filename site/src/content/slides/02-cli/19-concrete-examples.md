## Concrete Examples

### Browser Way
```
1. Copy code from VS Code
2. Paste into ChatGPT
3. Ask for changes
4. Copy response
5. Paste back into VS Code
6. Hope it works
7. If not, repeat
```

### CLI Way
```
1. "Fix the bug in src/api/handler.go"
2. AI reads file, makes changes, runs tests
3. Done (or iterate in place)
```
