## Practical Security Habits

- **Review before accepting** -- read AI-generated changes before committing
- **Never paste credentials** -- no API keys, tokens, or passwords into prompts
- **Use `.gitignore`** -- keep secrets files, `.env`, and credentials out of version control
- **Know what leaves your machine** -- AI sends your prompt text to the model provider; local files stay local unless you include them
- **Start small** -- grant minimal permissions, expand only as needed

> Good security is just good habits. These apply whether you use AI or not.
