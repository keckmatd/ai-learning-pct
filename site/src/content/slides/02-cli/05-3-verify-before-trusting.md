## 3. Verify Before Trusting

**Don't:**
```
You: "Commit these changes"
[Commits without reviewing]
```

**Do:**
```
You: "Show me the diff before committing"
[Review, then approve]
```

Trust but verify. AI makes mistakes. Review matters.
