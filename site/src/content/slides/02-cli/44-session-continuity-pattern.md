## Session Continuity Pattern

**End of session:**
```markdown
## Session Notes - 2025-04-15

Completed:
- Batch upload endpoint
- Validation logic

In progress:
- Error handling for malformed CSVs

Next:
- Add progress tracking
- Write tests
```

**Start of next session:**
- AI reads instructions.md
- Sees where you left off
- Picks up in context
