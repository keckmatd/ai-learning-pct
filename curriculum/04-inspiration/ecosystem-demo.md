# Ecosystem Demo

**Time**: 5 minutes  
**Mode**: Demo only (you show, they watch)

---

## What You're Showing

The `keck_companion` system - a fully integrated AI workflow.

**Move fast.** This is inspiration, not instruction.

---

## Demo Script

### 1. The Big Picture (30 seconds)

> "What you built today is the starting point. Here's where it can go after months of iteration."

Show the directory structure briefly:
```
keck_companion/
├── obsidian/           ← Knowledge base
├── rag/                ← Retrieval system
├── patterns/           ← Reply templates
├── workflows/          ← Complex automations
└── integrations/       ← Connected services
```

---

### 2. RAG in Action (90 seconds)

> "Remember RAG - retrieval augmented generation? Here's what it looks like when you have your own knowledge base."

**Demo:**
```bash
# Query that uses RAG
"What did we decide about the claims API in the architecture review?"
```

**Show:** AI searches your notes, finds the meeting summary, answers with citations.

**Point out:** "This is MY knowledge base. Not the internet. My notes, my decisions, my context."

---

### 3. Obsidian Integration (60 seconds)

> "This connects to my Obsidian notes. When I write notes, they become AI-searchable."

**Quick show:**
- Obsidian window with notes
- How those notes feed into RAG
- AI referencing a specific note

**Point out:** "The knowledge accumulates. Every note makes the system smarter about my work."

---

### 4. Reply Patterns (60 seconds)

> "I have templates for common communications. Same idea as the PowerPoint template, but for emails, reviews, etc."

**Demo:**
```bash
"Write a PR review for [pull request] using my review pattern"
```

**Show:** Consistent format, your voice, your standards.

---

### 5. Workflow Orchestration (30 seconds)

> "And these can chain together. 'Prepare for my 1:1 with Sarah' triggers: check notes, review recent PRs, summarize open items, generate talking points."

**Show briefly** (or just describe if time is tight).

---

## Key Points to Land

1. **This took time to build** - months of iteration
2. **It started where you are** - templates and instructions
3. **It compounds** - every addition makes the whole better
4. **It's yours** - customized for how YOU work

---
