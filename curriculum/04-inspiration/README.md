# Part 4: The Promise

**Duration**: 10 minutes  
**Goal**: Inspire. Show what's possible. Leave them wanting more.

---

## Module Files

1. [ecosystem-demo.md](./ecosystem-demo.md) - Show the keck_companion system
2. [whats-next.md](./whats-next.md) - Resources and next steps

---

## Facilitation Approach

This is a **demo and teaser**, not a tutorial.

Move fast. Show the impressive stuff. Don't explain implementation details.

**Goal emotion:** "Wow, I want to build something like that."

---

## Key Messages

1. **This is buildable.** Not magic, just accumulated skills and patterns.
2. **You start small.** Templates → skills → systems → ecosystems.
3. **The harness grows with you.** What you built today is version 0.1.
4. **Help is available.** You're not figuring this out alone.

---

## Time Budget

- Ecosystem demo: 5 min (fast, impressive)
- What's next: 5 min (resources, support, homework)

---

## End Strong

The workshop's success is measured by:
- Do they feel capable? (They DID something)
- Do they feel excited? (They WANT to do more)
- Do they know next steps? (They know HOW to continue)

All three must hit.
