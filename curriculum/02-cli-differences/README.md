## Teaching Approach

This part answers: "I get the concepts, but why would I leave the comfortable browser?"

**Frame it as:**
- Browser = consumer experience (they manage it for you)
- CLI = builder experience (you manage it yourself)

Neither is wrong - they're for different purposes.

---

## Key Emotional Beat

By the end of this section, they should feel:
- "Oh, I can do MORE with CLI"
- "The browser was actually limiting me"
- "I want to try this"

NOT:
- "CLI is for nerds"
- "This seems complicated"
- "Browser is fine for me"
