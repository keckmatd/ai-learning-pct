## Facilitation Notes

**This is where things can go wrong.** Corporate machines, network restrictions, permissions.

### Have Ready:
- Pre-installed demo machine for fallback
- Screenshots of each step
- Troubleshooting guide for common issues
- Buddy system (pair people who get stuck with those who succeed)

### Time Management:
- Install: 10 min (hard stop - if not working, pair up)
- Cheatsheet: 5 min (quick, reference for later)
- Exercise 1: 10 min (guided, everyone together)
- Exercise 2: 10 min (more independent)

---

## Success Criteria

By end of Part 3:
- [ ] At least run ONE successful command in harness
- [ ] Know where to find the cheatsheet
- [ ] Completed at least one exercise (even if watching)
- [ ] Have a working reference to return to

---

## Backup Plan

If install fails for most people:
1. Demo on your machine (screen share)
2. They follow along conceptually
3. Give them detailed async setup instructions
4. Focus time on understanding over doing

Better to show success than watch everyone troubleshoot.
